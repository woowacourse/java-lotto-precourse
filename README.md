# 로또 게임
    (1) 게임을 실행한다.
    (2) 구매하는 금액을 입력받는다.
    (3) 구매가능한 로또 개수를 출력한다.
    (4) 랜덤으로 6개의 숫자를 생성한다.
    (5) 중복이 있는지 확인한다.
    (6) 숫자를 오름차순으로 정렬한다.
    (7) 구매 가능한 개수만큼 반복한다. (4~6)
    (8) 지난 주 당첨 번호를 입력받는다.
    (9) 지난 주 보너스 번호를 입력받는다.
    (10) 지난 주 당첨 번호의 쉼표를 제거하고, int형으로 변환한다.
    (11) 몇개의 숫자가 일치하는지 알아내기.
    (12) 보너스 번호가 일치하는지 알아내기.
    (13) 등수 알아내기.
            * 예외 : 반복문 전에 3등과 2등을 처리해야 한다.
    (14) 금액 알아내기.
    (15) 당첨 금액별로 카운트 하기.
    (16) 당첨 금액별로 카운트 내용 출력하기.
