# 로또

## 소개
   * 가상으로 로또를 사보고 지난주 로또 번호로 수익률을 구해보자

## 기능 목록

   * 사용자의 구입금액 입력받기
        * 1000원 미만인 경우 종료
        * 숫자가 아닌 경우 종료
        * 1000의 배수가 아닌 경우 종료
   
   * 로또 구입 장수 만큼 로또 번호 발급
   
   * 로또 번호 6개의 숫자를 , 로 구분해서 입력 받는다
        * 6개의 숫자가 아닌 경우 종료
        * 숫자가 아닌 경우 종료
        * 45 이하의 숫자가 아닌 경우 종료
 
   * 보너스 번호를 입력 받는다
        * 숫자가 아닌 경우 종료
        * 45 이하의 숫자가 아닌 경우 종료
   
   * 수익율 계산하기
