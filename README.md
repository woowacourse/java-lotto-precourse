기능 요구
---------
* 로또 게임 기능을 구현해야 한다. 규칙을 모르면 검색해 로또 규칙을 찾아본다.
* 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급 해야한다.
* 로또 1장의 가격은 1000원이다.
* 로또 당첨금액은 고정 되어있는것으로 가정한다.
* 수익률을 계산해 출력해야 한다
